Quick Anki

Personal data

Quick Anki uses names of your Anki decks to let you choose which decks to add a new card to. No personal data is sent anywhere.

Third party services

Quick Anki uses Chrome (Chromium) and browser's Web Storage API (localStorage) for storing user's chosen decks.

The website

This website doesn't collect or send any personal data.
